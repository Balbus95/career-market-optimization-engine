# Career Market Optimization Engine (Skill for Gemini CLI)

An expert-level career engineering and market analysis skill designed to transform raw professional data into high-impact, ATS-optimized, and ESG-aligned application materials.

## Features
- **Dynamic Market Intelligence:** Real-time adaptation to 2025/2026 recruitment trends.
- **Strategic Gap Analysis:** Uses the $P = (I 	imes F) - L$ formula to identify urgent upskilling priorities.
- **B-Corp/ESG Alignment:** Specialized positioning for high-growth, purpose-driven organizations.
- **AI-Human Equilibrium:** Engineered to pass ATS filters while maintaining a unique "Human Voice" to combat AI-content saturation.
- **Boolean Infiltration:** Pre-built advanced search strings for the "Hidden Job Market."

## Installation
If you are using Gemini CLI, you can install this skill by cloning this repository into your skills directory:

```bash
cd ~/.gemini/skills
git clone https://github.com/Balbus95/career-market-optimization-engine
```

## Usage
Activate the skill within Gemini CLI:
> "Activate career-market-optimization-engine and analyze my profile..."

## Directory Structure
- `SKILL.md`: The core logic and workflow.
- `references/market_data.md`: Deep technical specifications (ATS, LinkedIn, B-Corp).
- `assets/`: Placeholders for visual templates or icons.
- `scripts/`: Utility tools for skills mapping and data parsing.
